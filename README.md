# CampusFlow

CampusFlow is a full-stack academic task management system designed to help students organize their academic work efficiently. It allows users to create tasks, assign them to subjects, set priorities, and track deadlines through a clean and modern dashboard.

This project was developed as a college full-stack project to demonstrate frontend–backend integration, REST APIs, and database usage.

---

## Features

- Create, view, and manage academic tasks  
- Assign tasks to subjects  
- Set task priority (Low / Medium / High)  
- Set due dates for tasks  
- Persistent data storage using a database  
- Clean and responsive user interface  
- Full frontend ↔ backend communication  

---
